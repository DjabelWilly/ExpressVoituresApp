# Express Voitures App

## Description

Application de gestion et vente de véhicules d'occasions.  
Le concept : Les voitures sont achetées aux enchères, réparées, puis elles paraissent sur le site web sous forme d'annonces pour y être vendus.


## Fonctionnalités

- Afficher les voitures de l’inventaire
- Ajouter (Achat) de nouvelles voitures
- Enregistrer des réparations
- Modifier les annonces (photos, descriptions, ...)
- Publier des annonces de vente
- Enregistrer les ventes

## Accès et sécurité

- **Visualisation** : Tout utilisateur peut consulter les annonces.
- **Édition** : Seul l'admin peut ajouter, modifier et supprimer les informations.
- Développement suivant les bonnes pratiques de sécurité pour protéger le site.

## UI Design

- Interface simple et fonctionnelle respectant la maquette fournie.
- Uniformité des saisies (dates via calendrier, années de voiture valides entre 1990 et aujourd’hui)
- Champs correctement étiquetés (ex. : modèle dans champ modèle, marque dans champ marque)

## Accessibilité

- Suivi des recommandations WCAG pour une utilisation par le plus grand nombre
- Ergonomie et navigabilité optimales

## Technologies

- ASP .NET Core
- Entity Framework Core
- SQL Server
